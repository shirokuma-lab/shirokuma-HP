つーる

Visual Studio Code
Visual Studio Codeとは、マイクロソフト社が開発しているプログラミングを行うときに使う「エディタ」というソフトウェアです。
カスタマイズ性が高く、テーマやショートカット、環境設定を変更できるなど無料ツールとは思えないほどの利便性です。
Paiza
Paizaとは、ギノ株式会社は運営するITエンジニア専門の総合求職サイトです。
用意されている問題に回答することで自分のプログラミングスキルをチェックできます。
また、Paizaラーニングで講座を受講すればC言語だけでなくPython,Javaなどを幅広くまなぶことができます。
GitHub
GitHubとは、ソフトウェア開発プロジェクトのためのソースコード管理アプリです。
企業でチーム開発を行うときにも使われています。
このしろくまlabのホームページもGitHubを使用してチームで作成しました！
TechFUL
TechFULとは、444株式会社が提供している無料オンラインプログラミング学習サービスです。
初級,中級,上級の問題が用意されており、それらを解くことで強み弱みが詳しくわかります。
イベントのコーディングバトルに参加していい成績を残すと賞金がもらえることも...
